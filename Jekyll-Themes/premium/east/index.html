Optimized for Maximum Performance and Great User Experience

East is a clean, minimal, modern theme that focus on content for Jekyll. East is created for those who want to create a simple and beautiful online blog. East it is 100% responsive, that means it fits with all different kind of devices in different sizes.

Features

  • Clean and minimal design
  • Instant Search
  • MailChimp newsletter subscription form
  • Custom CSS (Sass Source Files)
  • Clean well commented code, easy to customize
  • Valid HTML5 code
  • Mobile first & 100% Responsive layout
  • Multi author support
  • Google fonts (Merriweather)
  • Evil Icons
  • Tags (No GitHub Pages support)
  • Disqus comment system
  • Post sharing
  • Posts Pagination
  • Online documentation
