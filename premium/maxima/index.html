Maxima is a clean, minimal, modern theme that focus on content for Jekyll. Maxima is created for those who want to create a simple and beautiful online blog. Maxima it is 100% responsive, that means it fits with all different kind of devices in different sizes.

Features

  • Clean and minimal design
  • Custom CSS (Sass Source Files)
  • Clean well commented code, easy to customize
  • Valid HTML5 code
  • Instant search
  • Mobile first & 100% Responsive layout
  • Multi author support
  • Google fonts
  • Evil Icons
  • Tags (No GitHub Pages support)
  • Disqus comment system
  • Post sharing
  • Posts Pagination
  • Online documentation
