Cartwheel is a responsive blog theme for Jekyll built using HTML, Sass, and jQuery with a clean and modern design.

The theme includes a built in section for a personal Instagram feed that’s ready to go with an easy to get API key (required by Instagram). There are sections for a portfolio, which is built using a slider, and a blog with both archive and tag views.

All aspects of the website are editable like colors easily being switched directly from the _config.yml file instead of having to dig through any code. Documentation is also included with the files.

Features:

  • Responsive layout
  • Clean and modern layout using a mobile first design
  • Scrolling animations
  • Google Fonts
  • Social Media icons through Font Awesome
  • Portfolio slider
  • Tag pages
  • Regular updates and theme support

Updates:

Cartwheel for Jekyll will be updated periodically to support additional features
