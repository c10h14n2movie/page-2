Documenter is a Jekyll site template for all kinds of Digital product authors. You can make your support forum, documentation site or any knowledge base website by using that template. Do you have an open source project on github, you may use that template without any hesitation.

Feature

  • Uniqye Homepage Version
  • Modern and Creative Design
  • Pixel Perfect Design
  • Fully Responsive
  • Well Layered & Grouped in a Logical Manner
  • Based on Bootstrap 1170px Grid System
  • Uses free Font Awesome
  • Uses free Fonts (Google Fonts)
  • Easy to Customize
  • Fully High Resolution Design
  • SEO friendly design
  • Static HTML5 Site Template
  • DWQA Plugin’s HTML Layout Integrated
  • Super Fast – Faster Load ( Less Than 1 Sec)
  • W3 Validated and Cross Browser Supported
  • Regular Updates
  • Offline Documentation

N. B. That is a PSD Template. All the pictures are used in that template only for Preview Purpose. These are not included in PSD. You may get the pictures from Pixabay (FREE)
