Optimized for Maximum Performance and Great User Experience

Aspire is a modern, clean, and content and image focus theme for Jekyll. Aspire is created for those who want to create a beautiful online magazine, or news website. Aspire it is 100% responsive, that means it fits with all different kind of devices in different sizes.

Features

  • Clean and minimal design
  • Custom CSS (Sass Source Files)
  • Clean well commented code, easy to customize
  • Valid HTML5 code
  • Instant search
  • Mobile first & 100% Responsive layout
  • Sidebar with 6 Types of Widgets
  • Instagram Feed
  • Twitter Feed
  • Evil Icons
  • Tags (No GitHub Pages support)
  • Disqus comment system
  • Post sharing
  • Posts Pagination
  • Online documentation
